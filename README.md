<h1>Дипломная работа</h1>

## Команды для разворота

>## Всё указано с учётом докера (команда Sail)

>## Запуск и дроп контейнеров

```bash
  sail up -d
```

```bash
  sail stop
```

```bash
  sail down
```

>## Генирация, и миграции и установка

```bash
  sail artisan key:generate
```

```bash
  sail artisan migrate --seed
```

```bash
  sail npm i
```
>### Запуск vue.js


```bash
  sail npm run dev
```
